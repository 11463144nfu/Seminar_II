# 具冒充者與欺騙攻擊偵測之語音辨識
## Presntation by: 許正欣

一、 語者辨識的基本分類系統依據對語句內容的限制分為兩大類：
- 本文相關 (Text-dependent)： 系統限制使用者必須說出固定的文句內容 。雖然辨識效果通常較佳，但模型訓練相對困難 。  
- 本文無關 (Text-independent)： 不限制使用者說什麼，單純根據聲音特徵判斷身分 。  

二、 系統架構：雙重防護機制為了應對攻擊，系統將辨識流程分為兩個主要階段 ：  
- 攻擊偵測階段： 先判斷輸入語音是否為假語音或欺騙攻擊（Spoofing Attack）。  
- 核心技術： 使用 CQCC（常數 Q 倒頻譜係數）作為特徵 。  
- CQT 特色： 頻率軸採對數標度（Log scale），且視窗長度會隨頻率改變 。  

語者辨識階段： 通過欺騙偵測後，才進一步確認是否為目標語者 。  
- 核心技術： 使用 i-vector 特徵，其基於聯合因子分析（JFA）。  
- i-vector 用途： 將不同長度的語音轉為固定長度（400 維）的特徵，降低複雜度並利於分類器判斷 。  

三、 優化辨識準確度的策略演講中提到三種提升系統強健性的方法：
- 分數正規化 (Score Normalization)： 目的在於拉開目標語者與冒充者的分數分佈，使其更容易設定判斷門檻並降低誤判率 。  
- 條件式重試機制 (Conditional Retry)： 針對「錯誤拒絕（False Rejection）」問題，若系統拒絕語者，在特定條件下允許重新辨識 1~2 次，以提高目標語者的接受率 。  
- 全數決整體學習 (Unanimity-voting ensemble)： 結合多種分類器（如 Linear SVM, Kernel SVM, DNN 等），要求全數通過以提高對冒充者的拒絕率 。  

四、 實驗結果與結論根據模型實驗與測試數據：
- 分類器表現： 在欺騙攻擊偵測中，Single-model DNN 表現最優，正確率達 91.6%，且偽陽率（FPR）最低（10.8%）。  
- 模型架構： 隱藏層結構為 [9000, 32, 16, 8, 1] 的模型表現較佳 。  
- 整合效益： 單靠「欺騙攻擊偵測模型」對冒充者的阻擋率極低（僅約 1.7%），必須將其與「語者辨識模型」整合後，才能顯著提高整體系統的安全性與阻擋能力 。
