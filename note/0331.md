# 如何打造台版 Edge Impulse TinyML 開發平台
## Presentation by 許哲豪

今天的主題主要是說 邊緣運算在不同的情境中可大可小
從raspberry pi 到 server cluster都是可以被定義為edge compute的架構

也因為各種情形下都可以有AI介入 因此怎麼在有限的資源與空間中放入AI也是目前要關注的問題
今天提到的TinyML則主要是在MCU raspberry pi與arduino等小型裝置中使用AI的類型
使用 `Edge Impulse` 提供的環境可以簡易的完成從資料蒐集到訓練部屬等一連串的行為

並且可以快速的部屬到裝置中  讓學生與企業更容易地導入AI
並在最後提到AI到底該要怎麼應用的部分
不是盲目的全部相信與套用 而是必須要了解塔給出來的內容是否是正確的
